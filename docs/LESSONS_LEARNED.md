# Lessons Learned

## What Worked Well
- File-based HITL approval is simple and reliable
- Obsidian vault as dashboard gives full visibility
- Ralph Loop prevents Claude from stopping mid-task

## Challenges
- WhatsApp Web automation is fragile (UI changes break selectors)
- Instagram requires an image for every post
- Odoo authentication needs token refresh handling

## Improvements for Next Version
- Add A2A (Agent-to-Agent) messaging
- Move to cloud VM for 24/7 operation (Platinum Tier)
- Add SMS notifications for urgent approvals

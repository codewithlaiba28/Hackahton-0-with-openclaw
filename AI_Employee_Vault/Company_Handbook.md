# Company Handbook — Rules of Engagement

## Communication Rules
- Always be polite and professional
- Respond to client messages within 24 hours
- Flag any payment over $500 for human approval

## Email Rules
- Mark urgent if subject contains: invoice, payment, urgent, ASAP
- Auto-draft replies only for known contacts

## File Rules
- Never delete files, only move to /Done
- Log every action taken

## Escalation Rules
- Any unknown sender → flag for human review
- Any financial action → require approval file in /Pending_Approval
